# MyBookApp - Book Discovery Platform

A React application for searching books using Open Library API. Built as ALX Frontend Capstone.

## Features
- Search books by title/author/keyword
- Trending books on homepage
- Book details with cover, description, subjects
- White & blue theme with responsive design
- Pagination for browsing multiple pages

## Tech Stack
- React, Tailwind CSS, React Router
- Axios for API calls
- Open Library API
